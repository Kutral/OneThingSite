# OneThing Site

Static GitHub Pages landing page and privacy policy for OneThing, a calm local-first Android productivity app.

## Files

- `index.html` - Marketing landing page.
- `privacy.html` - Privacy policy for the Android app.
- `styles.css` - Shared visual system and responsive layout.
- `script.js` - Progressive reveal and pointer glow enhancements.

## Publish With GitHub Pages

Upload these files to a repository root, then enable GitHub Pages for the repository branch in GitHub settings. No build step is required.

The privacy policy uses the verified developer contact from the Google Play listing instead of a hard-coded support inbox.
